# Wata23 Packages

VRChat Creator Companion (VCC) 向けのパッケージ一覧です。  
このリポジトリを VCC に追加することで、配布中のパッケージをプロジェクトへ導入できます。

## 追加方法

リンクでもいいならこちらを押すことでVCCを起動して追加できます。心配な方はリンクより下の手動の手順でどうぞ。

[VCCに追加](vcc://vpm/addRepo?url=https%3A%2F%2Fwata23.github.io%2Fwata23-packages%2Findex.json)

1. VCC を開きます
2. `Settings` → `Packages` → `Add Repository` を開きます
3. 次の URL を追加します

    `https://wata23.github.io/wata23-packages/index.json`

4. 対象プロジェクトの `Manage Project` を開きます
5. 必要なパッケージを選んでインストールします

## 注意事項

- 古いバージョンは、互換性維持のため残す場合があります
- 各パッケージの使い方、更新内容、ライセンスの詳細は、個別リポジトリを確認してください
- パッケージ更新後、VCC やブラウザ上で最新情報がすぐに見えない場合は、キャッシュの影響で反映が遅れている可能性があります

## ライセンス

このリポジトリ自体は、各パッケージへの入口をまとめたものです。  
個別パッケージのライセンスは、それぞれのリポジトリおよびパッケージ内のライセンス表記を参照してください。
